How to use OpenVision web application

Instructions

  • Before you begin, place the video you wish to analyze in the video folder in this directory.
    If you have multiple videos shot from the same camera/position you only need one for now.

  • Click on the 'LOAD NEW PROJECT' button and select the video from the previous step.
    This will load the video onto the page so that you can accurately place the zone lines.

  • Next click on the 'ADD ZONE' button.
    This will initiate the drawing function in order to draw zone delineations.
    To draw the zones, start by holding down the left mouse button in the starting location.
    Move your cursor to the end location and release the mouse button.
    Repeat these steps for the remainder of the zones.

  • Next click on the 'ADD MASK' button.
    This step is optional, but will help the accuracy and speed of the tracker.
    To add a mask, left click and release at each vertex for the shape you wish to create. Try to add the vertices in a clockwise order.

  • Next define the directions.
    Click on the highlighted drop down box for 'NBR'. This will auto populate the rest of the movements based on your selection. This is not always accurate so make sure to check ALL the drop down boxes before moving on.

  • Next save your work.
    Click on the 'SAVE DETECTION ZONES' button. This will save a few .json files containing the data of your zones, mask, etc. You need to click 'allow multiple downloads' when the popup alert appears in the browser.

  • Finally, we can run the tracker and get our Turning Movement Counts.
    Click on the 'RUN TRACK' button. This will take a while, so feel free to come back at a later time to check the progress.
